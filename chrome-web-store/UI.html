三步开始用

权限克制：仅 storage、alarms，GitHub 域名按需授权。

1

配置仓库

在 Options 填写 Token / Owner / Repo，并授权 api.github.com 与 raw 域名。

2

同步到本地

手动同步或后台定时增量；JSON 与图片写入 IndexedDB。

3

发布页导入

打开小红书图文发布页，侧栏选内容，一键填入标题、正文与图片。
